## Descripción

Aplicación desarrollada con HTML, CSS y JavaScript puro. Incluye una calculadora estándar y un panel desplegable de funciones científicas mediante botón chevron, mejorando la experiencia del usuario y la escalabilidad del sistema.

---

## Características principales

| Funcionalidad       | Descripción                            |
| ------------------- | -------------------------------------- |
| Operaciones básicas | Suma, resta, multiplicación y división |
| Panel científico    | Funciones avanzadas desplegables       |
| Paréntesis          | Expresiones complejas                  |
| Historial           | Registro de operaciones                |
| Resultado en vivo   | Cálculo automático                     |
| Tema visual         | Claro y oscuro                         |
| Teclado físico      | Compatible                             |
| Responsive          | Móvil, tablet y escritorio             |
